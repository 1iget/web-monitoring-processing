web_monitoring.internetarchive.timestamped_uri_to_version(dt, uri, *, url, agency, site, view_url=None)[source]¶

Parameters:

dt : datetime.datetime

capture time

uri : string

URI of version

url : string

page URL

agency : string

primer metadata (likely to change in the future)

site : string

primer metadata (likely to change in the future)

view_url : string, optional

The archive.org URL for viewing the page (with rewritten links, etc.)
Returns:

dict : Version

suitable for passing to Client.add_versions
